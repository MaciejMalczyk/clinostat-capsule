# Clinostat capsule

Unified repository for a dispersed clinostat system. Each submodule/dir has its own Readme.

## Requirements

Each submodule has its own requirements. Additionally you need to install MongoDB database and MongoDB Compass for inspecting the data. The database should run alongside a Docker container for images on the same device.

Wireguard VPN needs public endpoint where its network will be configured. The easiest way to setup and manage such endpoint is wireguard-install script:

https://github.com/angristan/wireguard-install
